## Improvements to Hardware
If I was to make a V2 design for the hardware, I would make the following changes:
- Improved capacitor grounding
- Removing the low-current fuse for the 3.3V subsystem
- Adding a battery power option instead of a barrel jack
- Adding an enclosure to protect the board
- Moving the buttons to the center of the board under the screen for easier use
- Reducing annular widths to meet the size of JLC and other standard PCB fabs as opposed to Peralta's specs to create smaller boards
- Switching to SMD components to reduce the footprint of the board
- Sourcing more hot-swappable components to reduce supply chain issues
- Removing some extranous test points
- Adding a controllable debug LED, not just passive ones for communications
